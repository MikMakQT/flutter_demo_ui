# DnD Item Maker

DnD Item Maker is a Flutter application for creating, managing, and printing custom Dungeons & Dragons magic items.

---

## Features

The app allows users to:

- Create items from scratch  
- Import items from the D&D 5e SRD API  
- Store items in the cloud  
- Share items with other users  
- Export items as printable cards (PDF)  

### Core Features

-  Firebase Authentication (login/register)  
-  Create custom DnD magic items  
-  Live preview of item cards  
-  Cloud storage using Firestore  
-  Edit and delete items  
-  Import items from SRD API  
-  Share items between users  
-  Export and print items as PDF cards  

---

##  Technologies Used

- Flutter (UI framework)  
- Firebase (Authentication + Firestore)  
- D&D 5e SRD API (external data source)  
- PDF & Printing packages
